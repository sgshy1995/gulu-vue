# 开始安装

::: warning 警告
请确保你使用的版本是 Vue 2.6.0+。
:::

::: tip 提示
推荐你在项目中全部使用 `yarn` 来代替 `npm`。
:::

## 全局安装

如果你只是想尝试一下 Eden-Gulu，那么我们推荐你可以进行全局安装。

```sh
# 全局安装
yarn global add eden-gulu # 或者： npm install global eden-gulu
```

## 项目安装

在你的项目中使用 Eden-Gulu。

```sh
# 项目安装
yarn add -D eden-gulu # 或者： npm install -D eden-gulu
